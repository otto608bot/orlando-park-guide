# Orlando Park Guide

A family-friendly guide to Orlando theme park rides with filters for height requirements, pregnancy safety, wheelchair accessibility, and sensory considerations.

## Features

- **Height Filter**: Find rides your child can enjoy
- **Pregnancy-Safe**: No jarring movements or intense effects
- **Wheelchair Accessible**: Rides with no transfer required (WAV)
- **Sensory-Friendly**: No strobes, loud noises, or enclosed spaces

## Parks Covered

### Walt Disney World
- Magic Kingdom
- EPCOT
- Hollywood Studios
- Animal Kingdom

### Universal Orlando
- Universal Studios Florida
- Islands of Adventure

## Tech Stack

- Vanilla JavaScript (ES6 modules)
- Custom CSS (no frameworks)
- Mobile-first responsive design

## Development

```bash
# Serve locally
python3 -m http.server 8080

# Or use any static file server
npx serve .
```
