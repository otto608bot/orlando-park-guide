# Template: Daily workforce Telegram note

Keep short. Prefer links over prose.

```
PYP daily — YYYY-MM-DD

Done:
- …

Pipeline now:
- slug — stage — https://…

NEED APPROVAL (only if any):
- Decision: …
- Why: …
- Live/preview: https://…
- Reply: APPROVE / REJECT / EDIT …

Tomorrow (unless redirected):
- …
```

If nothing needs Bennett: end with `No action needed.`
