## Pin Design Templates

### Template 1: List Pin (Best for blog posts)
```
Size: 1000x1500px (2:3 ratio)

Layout:
- Top 20%: Title in bold, readable font
- Middle 60%: 3-5 list items with icons
- Bottom 20%: Website URL + branded color bar

Colors:
- Background: White or light cream
- Accent: Disney blue (#4A9DE8) or orange (#F37021)
- Text: Dark gray (#2D3748)

Font: Nunito (matches site branding)
```

### Template 2: Infographic Pin
```
Size: 1000x2100px (long)

Layout:
- Header: Post title
- Sections: 3-4 key points with icons
- Footer: Call to action + URL

Use for:
- Packing lists
- Height requirements
- Park comparisons
- Money-saving tips
```

### Template 3: Single Image Pin
```
Size: 1000x1500px

Layout:
- Full bleed image (hero image from post)
- Text overlay at top (title)
- Subtle gradient overlay for readability
- Logo/URL at bottom
```

### Formula:

## Content Calendar (Weekly)

### Weekly Pinning Schedule

| Day | Action | Count |
|-----|--------|-------|
| Monday | Pin new blog post | 3-5 pins per post |
| Tuesday | Repin popular content | 10 pins |
| Wednesday | Pin to group boards | 5 pins |
| Thursday | Create fresh pins for old posts | 5 pins |
| Friday | Repin trending content | 10 pins |
| Weekend | Engage (comments, follows) | 15 min/day |

### Monthly Content Mix
- 40%: Own blog content (new pins)
- 30%: Repins from relevant boards
- 20%: Fresh pins for old blog posts
- 10%: Group board contributions
