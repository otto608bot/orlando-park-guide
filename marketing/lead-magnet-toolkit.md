## 1. The Ultimate Packing Checklist

### Essentials (Don't Forget!)
- [ ] MagicBands or tickets (digital or physical)
- [ ] ID for adults
- [ ] Phone + portable charger
- [ ] Sunscreen (SPF 30+)
- [ ] Sunglasses + hat
- [ ] Comfortable walking shoes (2 pairs)
- [ ] Light jacket (indoor AC is cold)
- [ ] Rain poncho or small umbrella
- [ ] Refillable water bottle
- [ ] Snacks for the parks
- [ ] Autograph book + pen
- [ ] Small backpack or park bag

### For Kids
- [ ] Stroller (or rent at park)
- [ ] Change of clothes
- [ ] Wet wipes + hand sanitizer
- [ ] Favorite small toy/comfort item
- [ ] Glow sticks/light-up toys (for night)
- [ ] Kid-safe headphones

### Nice to Have
- [ ] Cooling towel (Florida heat!)
- [ ] Misting fan
- [ ] Blister bandages
- [ ] Ziploc bags (for wet clothes/phones)
- [ ] Portable phone fan
- [ ] External battery pack (20,000mAh+)

### What NOT to Bring
- Selfie sticks (banned)
- Wagons (banned)
- Large coolers
- Weapons of any kind
- Glass containers
- Alcoholic beverages

## 2. 7-Day Itinerary Template

### Day 1: Arrival Day
- [ ] Check into hotel
- [ ] Grocery run (snacks, water, breakfast items)
- [ ] Disney Springs for dinner + shopping
- [ ] Early bedtime (you'll need it!)

### Day 2: Magic Kingdom
- [ ] Rope drop (arrive 30 min before opening)
- [ ] Priority rides: Peter Pan, Seven Dwarfs, Space Mountain
- [ ] Afternoon break (2-4pm)
- [ ] Evening: Parade + fireworks

### Day 3: EPCOT
- [ ] Morning: Future World rides
- [ ] Afternoon: World Showcase
- [ ] Dinner: International pavilion
- [ ] Evening: Harmonious (if running)

### Day 4: Rest Day / Pool Day
- [ ] Sleep in
- [ ] Resort pool
- [ ] Disney Springs or mini golf
- [ ] Early prep for next park day

### Day 5: Hollywood Studios
- [ ] Priority: Galaxy's Edge (rope drop)
- [ ] Genie+ strategy for Tower of Terror, Rock 'n' Roller Coaster
- [ ] Shows: Indiana Jones, Frozen Sing-Along
- [ ] Evening: Fantasmic!

### Day 6: Animal Kingdom
- [ ] Morning: Avatar Flight of Passage (rope drop)
- [ ] Kilimanjaro Safaris (best before 10am)
- [ ] Afternoon: Festival of Lion King, Navi River
- [ ] Evening: Closing early, dinner at resort

### Day 7: Departure / Bonus Day
- [ ] Sleep in
- [ ] Revisit favorite park OR
- [ ] Disney Springs for souvenirs
- [ ] Airport transfer

## 4. Genie+ Cheat Sheet

### What Is It?
Paid service ($15-35/person/day) to skip lines at select attractions

### How It Works
1. Buy at midnight the night before (or day-of at 7am)
2. Book first Lightning Lane at 7am
3. Book next one after scanning into current ride OR after 2 hours
4. Repeat all day

### Best Value Rides (Book These First)
**Magic Kingdom:**
- Peter Pan's Flight (always longest wait)
- Jungle Cruise (afternoon)
- Big Thunder Mountain

**Hollywood Studios:**
- Slinky Dog Dash
- Tower of Terror
- Rock 'n' Roller Coaster

**EPCOT:**
- Test Track
- Soarin'
- Frozen Ever After

**Animal Kingdom:**
- Kilimanjaro Safaris (morning)
- Na'vi River Journey

### Pro Tips
- Set alarm for 6:55am
- Have park reservation confirmed
- Book highest priority ride first
- Don't book return times after 4pm (harder to get next one)
- Individual Lightning Lanes (separate purchase) for:
  - Galaxy's Edge rides
  - Avatar Flight of Passage
  - Seven Dwarfs Mine Train

## 7. Day-of Trip Checklist

### Morning (Before Leaving Hotel)
- [ ] Check weather forecast
- [ ] Charge phones + portable batteries
- [ ] Pack snacks and water
- [ ] Apply sunscreen
- [ ] Check park hours (can change!)
- [ ] screenshot Genie+ reservations
- [ ] screenshot dining reservations

### Park Bag Essentials
- [ ] Tickets/MagicBands
- [ ] ID
- [ ] Phone + charger
- [ ] Sunscreen
- [ ] Sunglasses
- [ ] Poncho (afternoon summer storms)
- [ ] Small snacks
- [ ] Water bottle
- [ ] Autograph book
- [ ] Basic first aid (bandages, pain reliever)

### At the Park
- [ ] Take photo of parking spot/row
- [ ] screenshot wait times at 11am, 2pm, 5pm
- [ ] Set next Genie+ booking alarm
- [ ] Hydrate every hour (Florida heat!)
- [ ] Take breaks (sit down every 2-3 hours)
- [ ] Photo pass photographers (free with Memory Maker)

### End of Day
- [ ] Check for lost items
- [ ] Note favorite moments
- [ ] Plan tomorrow's priority
- [ ] Charge everything
- [ ] Early bedtime!
